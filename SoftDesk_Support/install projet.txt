creer environnement vir: 
pipenv shell

Utilisez la commande pipenv lock pour générer le Pipfile.lock :

pipenv lock
Ensuite, utilisez la commande pipenv run avec pip freeze pour afficher les dépendances dans le terminal :

pipenv run pip freeze
Enfin, redirigez la sortie de la commande pip freeze vers un fichier requirements.txt :

pipenv run pip freeze > requirements.txt
Cela créera un fichier requirements.txt contenant les dépendances spécifiées dans le Pipfile.lock. Assurez-vous d'exécuter ces commandes dans le répertoire de votre projet où se trouve le Pipfile.lock.
